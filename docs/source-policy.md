# Source Policy

## Goal

Keep the repo useful as a trusted community asset, not a link farm.

## Preferred sources

1. Official websites, docs, changelogs, pricing pages.
2. Official GitHub, Hugging Face, ModelScope, OpenXLab, Paddle AI Studio.
3. Technical reports, arXiv papers, model cards.
4. Trusted benchmark sources: OpenCompass, C-Eval, SuperCLUE, CMMLU, MTEB/C-MTEB, BFCL, LongBench.
5. Credible technical media and reproducible community projects.

## Community-only sources

Social posts, screenshots, group chat messages, and short videos can be used as discovery signals, but stable entries need a more durable source.

## Hard blocks

- No reachable source URL.
- Unverifiable claims.
- Exact duplicate.
- Not related to Chinese AI models or ecosystem.
- Spam, SEO farm, or pure advertising.
- Unsafe or abusive playbooks.
- Random model mirrors without provenance.

## Benchmark claims

Record:

- benchmark name
- model version
- date
- source URL
- source type: official / third-party / community reproduction
- settings if known: zero-shot, few-shot, CoT, tool-use, search, API snapshot

Do not rewrite official marketing claims as neutral fact.
