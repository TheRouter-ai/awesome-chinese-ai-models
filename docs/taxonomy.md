# Taxonomy

This project tracks Chinese AI models, capabilities, ecosystem resources, and daily updates.

## Model categories

- `text_llm` — chat, instruction following, general text generation.
- `reasoning` — math, logic, chain-of-thought, complex problem solving.
- `coding` — code generation, completion, repair, software engineering.
- `vision_language` — image/document/chart/video understanding.
- `image_generation` — text-to-image and image editing.
- `video_generation` — text/image-to-video and video editing.
- `audio_speech` — ASR, TTS, voice cloning, music/audio generation.
- `embedding` — dense retrieval embeddings.
- `reranker` — reranking / cross-encoder models.
- `agent_tool_use` — function calling, tool use, browser or workflow agents.
- `long_context` — long-context and document understanding.
- `edge_small_model` — on-device or compact models.

## Verification status

- `verified` — backed by official or high-trust source.
- `unverified` — plausible but not confirmed enough for stable lists.
- `disputed` — conflicting information exists.
- `outdated` — likely stale or superseded.
