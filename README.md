# awesome-chinese-ai-models

A curated index of leading AI models developed in China, maintained by TheRouter.ai.

## What this repo provides
- A structured list format you can extend
- Links to official sources and licensing where available
- Notes (status, open-source, access) for discovery

## How to contribute
- Open an issue or PR with the model name, provider, and official link
- Include licensing info when possible
- Keep entries to official sources (avoid random mirrors)

## Disclaimer
This is a community-maintained index. Always verify details with the official provider.

---

We plan to later add a standard metadata schema (e.g., JSON/YAML) to make this index machine-readable.
