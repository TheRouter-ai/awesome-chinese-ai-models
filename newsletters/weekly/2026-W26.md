## New Models & Releases

**Tencent Hunyuan HY-Embodied** — Tencent's Hunyuan team open-sourced an embodied foundation model family designed for real-world agents. The umbrella repo ([HY-Embodied](https://github.com/Tencent-Hunyuan/HY-Embodied)) covers the full stack, while a dedicated [Hy-Embodied-0.5-VLA](https://github.com/Tencent-Hunyuan/Hy-Embodied-0.5-VLA) repo provides a Vision-Language-Action model bridging perception to robot control. Two Hugging Face checkpoints were published on day one:
- **[Hy-Embodied-0.5-VLA-RoboTwin](https://huggingface.co/tencent/Hy-Embodied-0.5-VLA-RoboTwin)** — fine-tuned for bimanual manipulation tasks in the RoboTwin benchmark (imitation learning).
- **[Hy-Embodied-0.5-VLA-UMI](https://huggingface.co/tencent/Hy-Embodied-0.5-VLA-UMI)** — pre-trained on UMI data for broader generalization.

Both reference arxiv:2606.14409. This is Tencent's first public release in the embodied AI / robotics space and positions them alongside teams like Physical Intelligence and Agibot.

## Tooling & Ecosystem

**Qwen Code v0.18.1 → v0.18.4** — The Qwen CLI coding assistant had an unusually active week with four stable releases plus multiple nightlies. Highlights across the sprint include gating direct-send in daemon mode, fixing CLI stop-after-cancel behavior, tracking supported server capabilities, and various core stability patches. The paper, "Beyond the Current Observation," targets scenarios where agents must reason under partial observability and stochastic state transitions.
